# homebrew-market-cli

Homebrew tap for market-cli, a stock portfolio tracking CLI application.

## Installation

```bash
brew install darul75/market-cli/market-cli
```

## Usage

```bash
market-cli
```

## Development

To update the formula when a new version is released:

1. Get the SHA256 of the new release binary:
   ```bash
   sha256sum path/to/market-cli
   ```

2. Update the formula with the new URL and SHA256

3. Commit and push changes
